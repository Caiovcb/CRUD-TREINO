O que é Git?
R:um lugar onde devs ajustam, modificam o codigo e etc

Qual é a diferença entre Git e GitHub?
R:git é onde os devs modificam o codigo e tec , git hub é onde as modificacoes sao mostradas

O que o comando git commit -m faz?
R: deixa um comentario sobre as atualizacoes feitas

Como criar uma nova branch em Git?
R:git branch nome_da_branch
git checkout -b nome_da_branch

Qual é o comando para enviar suas alterações para o GitHub, Gitea, bitbucket....etc?
R:git push

Como você clona um repositório do GitHub para sua máquina?
R:git clone https://github.com/usuario/nome_do_repositorio.git

O que é um "pull request"?
R:é um "pedido para outras pessoas verem seu codigo"

o que faz o arquivo .gitignore?
R:e um arquivo que ignora tudo dentro dele na hora de subir algum projeto

teste o gitignore adicionado algum arquivo e subindo para o repositorio para ver se funciona.
